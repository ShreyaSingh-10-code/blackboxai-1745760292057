# Dream Guardian

## Project Overview
Dream Guardian is a web application designed to help users set and track their personal goals while providing motivational guidance. The application offers an intuitive interface where users can input their dreams and receive suggestions on relevant educational paths, enabling them to work towards achieving their objectives.

## Installation
To run Dream Guardian locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd dream-guardian
   ```

2. **Open the index.html file in your web browser:**
   Simply double-click the `index.html` file or use a local server of your choice.

## Usage
- Open the application in your web browser.
- Use the "Set Goals" section to input your dream or goal.
- Submit the form to see your goal displayed below and receive educational suggestions tailored to your dream.
- Explore the "Motivational Guidance" section for tips and inspiration.
- Learn more about the application in the "About" section.

## Features
- Set and track personal goals.
- Receive motivational quotes and guidance.
- Get educational suggestions based on the user's dreams.
- Responsive design that works well on various devices.

## Dependencies
This project uses the following external libraries and frameworks:

- **Font Awesome**: For icons.
- **Google Fonts (Montserrat and Roboto)**: For typography.
- **Tailwind CSS**: For styling.

You can include them directly in your HTML as shown in the `index.html` file provided above.
